# ☸️ Karma Monitor (Hộ Pháp Auditor)

Karma Monitor is a real-time "karma" surveillance tool for the digital realm. Built with Go (Golang) to ensure "Diamond-Indestructible" performance and a tracking system that never misses a single push.

---

## ✨ "Guardian" Features
- **Real-time Polling:** Automatically stalks GitHub Public Events with ultra-low latency.
- **Deep Secret Scan:** Scans for "karma" (secrets, env, passwords) lurking in commit diffs using security-grade Regex.
- **Spiritual Persistence:** Stores the last event ID in a "Black Book" (state file) ensuring no drama is missed even if the system takes a nap.
- **Ò Í E Alert:** Instant notifications via Discord Webhook whenever "mime-clown" behavior is detected.

---

## 🛠️ "Omnipotent" Technologies
- **Language:** Go 1.25.4 (Optimized for M4 Pro).
- **Container:** Docker & Docker Compose (Total isolation from worldly dust).
- **Infrastructure:** Multi-stage Build (Light as a feather, only ~15MB).

---

## 🚀 How to "Summon" the Bot
- Clone this repo to your million-dollar machine.
- Edit `main.go` (Input username and Webhook in the "private zone").
- Initialize the black book: `mkdir state`.
- Deploy: `docker compose up -d --build`.
